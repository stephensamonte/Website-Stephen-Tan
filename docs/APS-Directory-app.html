Arlington Public Schools Directory

Functional Android & iOS Concept Application developed while working for Arlington Public Schools Summer 2018.

Android and iOS applicaiton was developed using Flutter and an Amazon Web Client.
The promotional video to assist the application through politics was made using Adobe Premiere Pro, After Effects, and Cinema 4d.
While working with Arlington Public Schools Summer 2018.

Problem:

- Directory Accessibility
- Up-To-Date Information

Features:

- Works on Majority of Smart Devices
+ All Android (API 16 and UP)
+ Samsung, Nexus, etc.
+ All iOS
- Search Bar (Intelligent Search) (Search by partial name, number, department)
- Quick Call or Text
- Saved Recent Searches
- Dynamic Data Access
- Modular Design (Future Expansion)
+ Directory
+ ELP
- Future Modules

Future Stuff:

- App can be dynamic
+ Modules can be added on to add functionality
+ Access to different data can be controlled
(Examples:
+ English Language Proficiency module
+ Exit ticket meeting feedback responses)
